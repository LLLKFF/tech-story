# US-China Tech Giants Data Story

## Project Overview
This interactive data visualization explores the comparative performance of US and Chinese technology giants from 2015-2024. Built with R, Quarto, and Plotly.

## Files Structure
- `tech_story.html` - Main interactive visualization
- `tech_story.qmd` - Source Quarto document
- `scripts/` - R scripts for data processing and visualization
- `data/` - Raw and processed stock data
- `styles.css` - Custom styling

## How to Run
1. Open `tech_story.html` in any modern web browser
2. Scroll through the interactive narrative
3. Hover over charts for detailed information

## Technical Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for interactive Plotly charts

## Data Sources
- Yahoo Finance via tidyquant R package
- Period: January 2015 - November 2024
